# Kotlin WebApp Skeleton

Written in Kotlin using the following base libs
- Netty (embedded async web server)
- Jersey (jax-rs)
- Jackson (json)
- Logback (logging)

## Build Project
```
mvn clean package
```

## Run Application
```
java -jar target/kotlin-netty-jaxrs-embedded-skeleton-1.0.0-SNAPSHOT.jar
```
